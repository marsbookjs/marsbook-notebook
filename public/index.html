Groq API Key Required

Connect a free Groq key to unlock the AI assistant — fast LLM inference at no cost.

  1. 1
    Create a free Groq account Visit console.groq.com and sign up — it's free, no credit card needed.
  2. 2
    Generate an API key In the Groq console, go to API Keys → Create API Key. Copy the key shown — you won't see it again.
  3. 3
    Add it to Environment Variables Click the 🔑 Env button in the left sidebar, add a variable named GROQ_API_KEY, paste your key as the value, then click Save.
  4. 4
    Reopen this panel Close and reopen the AI Assistant — it will detect your key automatically and be ready to use.
